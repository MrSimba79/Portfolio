About me

I'm Rodrigo and I'm a front-end developer.
I started my carreer by doing a 1 year long Full-Stack Web Development course in FLAG school, where I quickly laid to rest any doubts I had and fell in love with web development.
In this course I learned Agile Methodologies, such as SCRUM and Kanban, I learned about UI/UX and then to the more pratical: HTML, CSS, JavaScript, Typescript, Angular, Git, MySQL, MongoDB, PHP, NodeJS.
During this course I developed two projects, the first focused on front-end and the second focused on back-end.

After that, I had the opportunity to do a very intense 1 month Javascript and React Bootcamp at Upgrade Hub, and for my success in it I was able to join Minsait (an Indra Systems company) and participate at a project for Inditex.
During this project I was main tasks were:
- Develop and optimize features using Javascript, React with Redux and CSS.
- Monitoring app performance, bug fixes and code review.
- Maintaining an organized workflow using a project management tool (Git - Sourcetree).
- App testing using React Testing Library and Jest.
